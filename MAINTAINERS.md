# Maintainers

## Current Maintainer Team

- Primary maintainer: KonkovDV
- Repository: https://github.com/KonkovDV/AeroBIM

## Responsibilities

Maintainers are expected to:

1. preserve clean architecture boundaries and DI/port discipline;
2. keep cross-modal validation behavior deterministic where required;
3. ensure report provenance and contract compatibility remain explicit;
4. review security-sensitive changes (workflows, storage adapters, auth, dependencies);
5. keep README and governance files aligned with shipped behavior.

## Escalation

Security incidents follow SECURITY.md private disclosure process.
Release-critical decisions should be approved by at least one maintainer.
